Welcome to the intersection of Ethereum and Nix

Leverage the power of Ethereum and Nix, and transform your experience of running and operating Ethereum related services and infrastructure with Ethereum.nix

Get started

Run Ethereum applications without fuss

Every Ethereum application on Ethereum.nix is crafted from its source. This ensures code reproducibility, establishing trustworthiness. And with Nix, you get the flexibility to tweak the build process as minutely as you desire.

Instantly run applications like Prysm, Erigon, Foundry, Slither and ethdo, among many others. No matter what OS you're running as long as it's supported by Nix. Say goodbye to Docker for running applications, instead, run them natively.

Streamlined operations

Our tailor-made NixOS modules simplify your work running Execution and Consensus clients, but also making tasks like backups effortless. And that's not all. We're also gearing up to support Liquid Staking deployments and custom PoS networks, making development and testing even easier. Stay tuned for these exciting additions!
